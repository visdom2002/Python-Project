# Python-Project

This project analyzes Zomato restaurant data using Python (Pandas, Matplotlib, Seaborn) to explore restaurant types, ratings, 
pricing, online ordering trends, and customer engagement. Data cleaning, exploratory data analysis, and visualization 
techniques were used to extract meaningful business insights. 


Conclusion

The analysis shows that most restaurants fall in the affordable dining category with moderate ratings.
Online ordering adoption is increasing, while customer engagement varies across restaurants, highlighting competitive market dynamics.
